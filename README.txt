مشروع: واجهة شطرنج مموهة مع غرفة سرية للدردشة
إعداد: أحمد قراجة

المحتوى:
- index.html: الصفحة الرئيسية التي تحتوي على لوحة الشطرنج.
- style.css: تصميم الشطرنج الفاخر.
- secret-style.css: تصميم الغرفة السرية.
- chess.js: رسم قطع الشطرنج على اللوحة.
- utils.js: التحقق من تسلسل المفاتيح السري.
- secret.js: إظهار الغرفة السرية عند إدخال التسلسل.
- chat.js: نظام دردشة بسيط داخل الغرفة السرية.
- logo.png: شعار حزب تكافل، يظهر فقط داخل الغرفة السرية.

طريقة الاستخدام:
1. افتح index.html في المتصفح.
2. أدخل التسلسل التالي عبر لوحة المفاتيح:
   ↑ ↑ ↓ ↓ ← → ← →
3. ستفتح الغرفة السرية مع الشعار وخانة الدردشة.

⚠️ لا يتم تخزين أي رسائل، ولا توجد إشعارات أو إشارات مرئية خارج الغرفة السرية.
